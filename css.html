# 命名规范

  • 统一使用"-"连字符
  • 尽量使用小写字符
.news-title { ... }
  • 禁止使用id作为样式名称
// bad
#app { ... }

// good
.app { ... }
  • 对于要覆盖ui库的样式，我们约定用or-开头的class，并对应ui名来避免污染
// 覆盖一个element-ui的一个primary的button
.or-button-primary { ... }

# 样式约定

  • 省略值为 0 时的单位
// bad
padding-bottom: 0px;
margin: 0em;

// good
padding-bottom: 0;
margin: 0;
  • 使用Less/Sass时，避免使用嵌套，如嵌套，深度不大于3层。
  • 如没有兼容性要求，尽量采用 flex布局
